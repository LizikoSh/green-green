# Зелена кімната — демо-сайт про кімнатні рослини

## Файли

- `index.html` — структура сторінки;
- `style.css` — усі стилі та адаптивність;
- `script.js` — каталог, фільтри, акордеони, тест і нагадування поливу.

## Запуск

1. Завантажте всі три файли в одну папку.
2. Відкрийте `index.html` у браузері.
3. Для публікації завантажте папку на GitHub Pages, Netlify або Vercel.

## Що вже працює

- велике hero-фото;
- мобільне меню;
- каталог із пошуком та трьома фільтрами;
- інтерактивні блоки догляду та FAQ;
- календар поливу з локальним збереженням у браузері;
- тест підбору рослини;
- адаптація під телефон, планшет і комп’ютер.

## Важливо

Фото підтягуються з Unsplash через зовнішні URL. Для повністю автономного сайту завантажте потрібні зображення у локальну папку `images` і замініть URL у `index.html`, `style.css` та `script.js`.

Контактна форма є демонстраційною. Для реального надсилання повідомлень потрібно підключити Formspree, Netlify Forms, власний сервер або інший сервіс.
