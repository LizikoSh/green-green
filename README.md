# зелено-зелено. — багатосторінковий сайт про рослини

## Структура

- `index.html` — головна сторінка з повноекранним hero та меню 2×2;
- `plants.html` — довідник кімнатних рослин із пошуком і фільтрами;
- `interior.html` — галерея рослин в інтер’єрі;
- `accessories.html` — галерея пристосувань для догляду;
- `creativity.html` — галерея творчих робіт на тему рослин;
- `mistakes.html` — поширені помилки новачків;
- `style.css` — спільні стилі всіх сторінок;
- `script.js` — мобільна навігація та фільтрація довідника;
- `assets/hero-home.webp` — локальне hero-фото, надане користувачем.

## Запуск

Відкрийте `index.html` у браузері або опублікуйте всю папку на GitHub Pages, Netlify чи Vercel.

## Джерела зображень

- Каталог рослин: назви, фото, ціни та посилання на товарні сторінки Dflora.
- Галереї й навігаційні картки: зовнішні зображення Unsplash.

Перед комерційною публікацією варто перевірити умови використання фотографій Dflora або замінити їх на власні/ліцензовані файли. Ціни можуть змінюватися.

## Заглушки

Контактні дані й посилання на соціальні мережі у футері є демонстраційними та потребують заміни.
