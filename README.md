# зелено-зелено. — сайт про кімнатні рослини

Статичний адаптивний багатосторінковий сайт.

## Файли

- `index.html` — головна сторінка з повноекранним hero-фото і меню 2×2.
- `plants.html` — довідник кімнатних рослин.
- `interior.html` — фотогалерея рослин в інтер’єрі.
- `accessories.html` — галерея пристосувань.
- `creativity.html` — галерея творчих робіт.
- `mistakes.html` — помилки новачків.
- `style.css` — усі стилі сайту.
- `script.js` — мобільне меню, база рослин, пошук і фільтри.
- `assets/hero-home.webp` — локальне hero-зображення.

## База рослин

У довіднику 31 рослина, вибрана зі сторінок категорії «Кімнатні рослини» української Вікіпедії. Для кожної картки додано:

- українську назву;
- латинську назву;
- фотографію з Wikimedia Commons;
- короткий опис;
- посилання на відповідну статтю Вікіпедії;
- редакційні параметри для фільтрів: складність, умови, полив і обережність для дітей/тварин.

Ботанічні дані та фотографії походять із Вікіпедії/Wikimedia Commons. Позначки догляду й безпечності не є ветеринарною або медичною консультацією та потребують перевірки для конкретного виду.

## Запуск

Відкрийте `index.html` у браузері або розмістіть папку в корені статичного хостингу. Для Vercel із файлами в корені Output Directory має бути `.`.
